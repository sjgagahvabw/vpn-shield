# VPN Shield - Руководство пользователя

## Введение

VPN Shield - это продвинутая VPN система, специально разработанная для обхода блокировок в России. Система поддерживает несколько протоколов с автоматическим переключением.

## Поддерживаемые протоколы

### 1. REALITY (Рекомендуется)

**Преимущества:**
- Неотличим от обычного HTTPS трафика
- Маскируется под реальные сайты (Microsoft, Bing)
- Максимальная устойчивость к DPI
- Высокая скорость

**Когда использовать:**
- Основной протокол для WiFi
- Когда нужна максимальная скрытность
- Для обхода SNI filtering

**Клиенты:**
- v2rayN (Windows)
- v2rayNG (Android)
- Shadowrocket (iOS)
- Hiddify (все платформы)

### 2. Hysteria2 (Для мобильных сетей)

**Преимущества:**
- Оптимизирован для нестабильных соединений
- QUIC протокол (UDP)
- Отличная скорость на мобильных сетях
- Обходит TCP throttling

**Когда использовать:**
- Мобильный интернет (4G/5G)
- Нестабильное соединение
- Когда TCP заблокирован

**Клиенты:**
- Hiddify (все платформы)
- NekoBox (Android)
- Shadowrocket (iOS)

### 3. Trojan (Backup)

**Преимущества:**
- Простой и надежный
- TLS шифрование
- Хорошая совместимость

**Когда использовать:**
- Когда REALITY не работает
- Для старых устройств
- Как резервный вариант

**Клиенты:**
- v2rayN (Windows)
- v2rayNG (Android)
- Shadowrocket (iOS)

### 4. VMess (Универсальный)

**Преимущества:**
- Классический протокол V2Ray
- WebSocket поддержка
- Работает через CDN

**Когда использовать:**
- Для работы через Cloudflare
- Когда нужен WebSocket
- Универсальный вариант

**Клиенты:**
- v2rayN (Windows)
- v2rayNG (Android)
- Все V2Ray клиенты

## Установка клиентов

### Windows

**v2rayN (Рекомендуется)**

1. Скачайте с GitHub: https://github.com/2dust/v2rayN/releases
2. Распакуйте архив
3. Запустите v2rayN.exe
4. Импортируйте конфигурацию:
   - Через QR код: Сервер → Импорт из QR кода
   - Через URL: Сервер → Импорт из буфера обмена
   - Через файл: Сервер → Импорт конфигурации

**Hiddify**

1. Скачайте с https://hiddify.com
2. Установите приложение
3. Добавьте конфигурацию через QR код или ссылку

### Android

**v2rayNG (Бесплатный)**

1. Установите из Google Play или GitHub
2. Нажмите + в правом верхнем углу
3. Выберите "Импорт из QR кода" или "Импорт из буфера обмена"
4. Нажмите на конфигурацию для подключения

**Hiddify (Рекомендуется)**

1. Установите из Google Play
2. Нажмите "Add Profile"
3. Отсканируйте QR код или вставьте ссылку
4. Нажмите Connect

**NekoBox (Для Hysteria2)**

1. Установите из Google Play или GitHub
2. Добавьте профиль через QR или ссылку
3. Подключитесь

## Получение конфигурации

### Через веб-панель

1. Войдите в панель управления
2. Перейдите в "Конфигурации"
3. Выберите нужный протокол
4. Скопируйте ссылку или отсканируйте QR код

### Форматы конфигураций

**REALITY:**
```
vless://uuid@server:443?security=reality&sni=www.microsoft.com&fp=chrome&pbk=publickey&type=tcp&flow=xtls-rprx-vision#REALITY
```

**Hysteria2:**
```
hysteria2://password@server:36712?sni=server.com&obfs=salamander#Hysteria2
```

**Trojan:**
```
trojan://password@server:8444?sni=server.com&type=tcp#Trojan
```

**VMess:**
```
vmess://base64_encoded_json
```

## Настройка клиентов

### Оптимальные настройки для России

**Общие:**
- Режим маршрутизации: Proxy (не Global)
- DNS: 8.8.8.8, 1.1.1.1
- Mux: Включен (для VMess/Trojan)

**v2rayN/v2rayNG:**
```
Настройки → Параметры маршрутизации
- Режим: Обход LAN и материкового Китая
- Или создайте свои правила для российских сайтов
```

**Shadowrocket:**
```
Конфигурация → Правила
- Добавьте российские домены в DIRECT
- Остальное через PROXY
```

### Правила маршрутизации

Для доступа к российским сайтам напрямую, добавьте в правила:

```
# Российские домены - напрямую
domain:yandex.ru
domain:vk.com
domain:mail.ru
domain:gosuslugi.ru

# Заблокированные - через VPN
domain:twitter.com
domain:facebook.com
domain:instagram.com
domain:youtube.com
```

## Автоматическое переключение протоколов

### Hiddify (Рекомендуется)

Hiddify поддерживает автоматическое переключение между протоколами:

1. Добавьте все протоколы (REALITY, Hysteria2, Trojan)
2. Включите "Auto Select"
3. Приложение автоматически выберет лучший протокол

### Subscription (Подписка)

Создайте файл подписки со всеми протоколами:

```
vless://...#REALITY
hysteria2://...#Hysteria2
trojan://...#Trojan
vmess://...#VMess
```

Импортируйте URL подписки в клиент.

## Решение проблем

### Не подключается

1. **Проверьте интернет:**
   ```bash
   ping 8.8.8.8
   ```

2. **Проверьте порты:**
   - REALITY: 443 (TCP)
   - Hysteria2: 36712 (UDP)
   - Trojan: 8444 (TCP)
   - VMess: 8443 (TCP)

3. **Попробуйте другой протокол:**
   - WiFi → REALITY
   - Мобильный → Hysteria2
   - Если не работает → Trojan

### Медленная скорость

1. **Проверьте загрузку сервера** в панели управления
2. **Попробуйте Hysteria2** для мобильных сетей
3. **Отключите Mux** в настройках клиента
4. **Измените DNS** на 1.1.1.1

### Периодические разрывы

1. **Используйте Hysteria2** - он лучше работает с нестабильным соединением
2. **Включите Keep Alive** в настройках клиента
3. **Уменьшите MTU** до 1400

### Заблокировали протокол

1. **Переключитесь на другой протокол**
2. **Используйте Domain Fronting** (VMess через Cloudflare)
3. **Свяжитесь с администратором** для обновления конфигурации

## Советы по безопасности

1. **Не делитесь конфигурацией** с другими
2. **Используйте разные пароли** для разных сервисов
3. **Включите Kill Switch** в клиенте (если доступно)
4. **Проверяйте IP** после подключения: https://whoer.net
5. **Используйте HTTPS** везде, где возможно

## Проверка работы VPN

### Проверка IP адреса

```bash
# До подключения
curl ifconfig.me

# После подключения (должен измениться)
curl ifconfig.me
```

### Проверка DNS утечек

Откройте: https://dnsleaktest.com

Должны быть DNS серверы вашего VPN, а не провайдера.

### Проверка WebRTC утечек

Откройте: https://browserleaks.com/webrtc

Убедитесь, что не виден ваш реальный IP.

## Часто задаваемые вопросы

**Q: Какой протокол самый быстрый?**
A: Hysteria2 для мобильных сетей, REALITY для WiFi.

**Q: Можно ли использовать на нескольких устройствах?**
A: Да, один аккаунт работает на всех ваших устройствах (лимит устройств настраивается).

**Q: Что делать при отключении интернета?**
A: Используйте Hysteria2 - он лучше работает в условиях ограничений.

**Q: Безопасно ли это?**
A: Да, весь трафик зашифрован. REALITY неотличим от обычного HTTPS.

**Q: Логируется ли мой трафик?**
A: Нет, система не хранит логи трафика. Только статистика использования.
