# Contributing to VPN Shield

Спасибо за интерес к проекту VPN Shield! Мы приветствуем любой вклад.

## Как внести вклад

### Сообщение об ошибках

Если вы нашли ошибку:

1. Проверьте, не была ли она уже сообщена в [Issues](https://github.com/yourusername/vpn-shield/issues)
2. Если нет, создайте новый issue с подробным описанием:
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Фактическое поведение
   - Версия системы и окружение
   - Логи (если применимо)

### Предложение новых функций

1. Создайте issue с описанием функции
2. Объясните, зачем она нужна
3. Предложите возможную реализацию

### Pull Requests

1. Fork репозиторий
2. Создайте feature branch:
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. Внесите изменения
4. Добавьте тесты (если применимо)
5. Убедитесь, что код соответствует стилю проекта
6. Commit изменения:
   ```bash
   git commit -m 'Add amazing feature'
   ```
7. Push в branch:
   ```bash
   git push origin feature/amazing-feature
   ```
8. Откройте Pull Request

## Стиль кода

### Go
- Используйте `gofmt` для форматирования
- Следуйте [Effective Go](https://golang.org/doc/effective_go)
- Добавляйте комментарии к экспортируемым функциям

### TypeScript/React
- Используйте TypeScript для типизации
- Следуйте [Airbnb Style Guide](https://github.com/airbnb/javascript)
- Используйте функциональные компоненты и хуки

### Commit Messages
- Используйте настоящее время ("Add feature" не "Added feature")
- Первая строка - краткое описание (до 50 символов)
- Подробное описание в теле commit (если нужно)

Примеры:
```
Add REALITY protocol support
Fix user authentication bug
Update documentation for deployment
```

## Тестирование

Перед отправкой PR убедитесь, что:

1. Код компилируется без ошибок
2. Все существующие тесты проходят
3. Новый функционал покрыт тестами
4. Документация обновлена

## Лицензия

Внося вклад в проект, вы соглашаетесь с тем, что ваш код будет распространяться под лицензией MIT.

## Вопросы?

Если у вас есть вопросы, создайте issue или свяжитесь с нами.

Спасибо за вклад в VPN Shield! 🛡️
