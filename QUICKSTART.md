# VPN Shield - Быстрый старт

## За 5 минут до запуска

### 1. Подготовка сервера (1 минута)

```bash
# Подключитесь к вашему VPS
ssh root@YOUR_SERVER_IP

# Скачайте проект
git clone https://github.com/yourusername/vpn-shield.git
cd vpn-shield
```

### 2. Автоматическая установка (3 минуты)

```bash
# Запустите скрипт быстрого старта
chmod +x scripts/start.sh
sudo ./scripts/start.sh
```

Скрипт автоматически:
- ✅ Установит Docker и Docker Compose
- ✅ Создаст конфигурацию с безопасными паролями
- ✅ Запустит все сервисы
- ✅ Покажет учетные данные администратора

### 3. Доступ к панели (1 минута)

```bash
# Откройте в браузере
http://YOUR_SERVER_IP:8888

# Войдите с учетными данными из вывода скрипта
Username: admin
Password: (показан в выводе скрипта)
```

## Создание первого пользователя

### Через веб-панель

1. Войдите в панель управления
2. Перейдите в "Пользователи" → "Добавить пользователя"
3. Заполните данные:
   - Username: user1
   - Email: user1@example.com
   - Password: SecurePassword123
4. Выберите протоколы (рекомендуется все)
5. Нажмите "Создать"

### Через API

```bash
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "user1",
    "email": "user1@example.com",
    "password": "SecurePassword123"
  }'
```

## Получение конфигурации

1. В панели перейдите в "Конфигурации"
2. Выберите пользователя
3. Выберите протокол (REALITY для WiFi, Hysteria2 для мобильных)
4. Скопируйте ссылку или отсканируйте QR код

## Подключение клиента

### Android (v2rayNG)

1. Установите v2rayNG из Google Play
2. Нажмите + → "Импорт из буфера обмена"
3. Вставьте скопированную ссылку
4. Нажмите на конфигурацию для подключения

### iOS (Shadowrocket)

1. Купите Shadowrocket в App Store ($2.99)
2. Откройте приложение
3. Нажмите + → "Добавить из буфера"
4. Вставьте ссылку
5. Подключитесь

### Windows (v2rayN)

1. Скачайте v2rayN с GitHub
2. Запустите v2rayN.exe
3. Сервер → Импорт из буфера обмена
4. Вставьте ссылку
5. Подключитесь

## Следующие шаги

### Настройка домена (рекомендуется)

1. Купите домен (Namecheap, Cloudflare)
2. Создайте A записи:
   ```
   A    @         YOUR_SERVER_IP
   A    panel     YOUR_SERVER_IP
   ```
3. Получите SSL сертификат:
   ```bash
   apt install certbot -y
   certbot certonly --standalone -d your-domain.com
   ```
4. Обновите .env:
   ```bash
   nano /opt/vpn-shield/.env
   # Измените DOMAIN=your-domain.com
   ```
5. Перезапустите:
   ```bash
   docker-compose restart
   ```

### Настройка Cloudflare (опционально)

1. Добавьте домен в Cloudflare
2. Включите прокси (оранжевое облако)
3. SSL/TLS → Full (strict)
4. Получите API токен
5. Добавьте в .env:
   ```env
   CLOUDFLARE_API_TOKEN=your_token
   CLOUDFLARE_ZONE_ID=your_zone_id
   ```

### Оптимизация для России

1. **REALITY** - основной протокол
   - Маскируется под Microsoft/Bing
   - Неотличим от обычного HTTPS
   
2. **Hysteria2** - для мобильных
   - QUIC протокол (UDP)
   - Работает при нестабильном соединении
   
3. **Множественные конфигурации**
   - Создайте конфиги для всех протоколов
   - Используйте автопереключение в Hiddify

## Решение проблем

### Не подключается

1. Проверьте firewall:
   ```bash
   ufw status
   ufw allow 443/tcp
   ufw allow 36712/udp
   ```

2. Проверьте логи:
   ```bash
   docker-compose logs backend
   ```

3. Попробуйте другой протокол

### Медленная скорость

1. Используйте Hysteria2 для мобильных
2. Проверьте загрузку сервера в панели
3. Попробуйте другую локацию сервера

### Периодические разрывы

1. Переключитесь на Hysteria2
2. Включите Keep Alive в клиенте
3. Уменьшите MTU до 1400
